About the Rising Waters System

What is this project?

Rising Waters is an early flood warning system that leverages meteorological data to proactively predict the likelihood of floods. Our goal is to assist emergency services and local communities by providing crucial advanced warnings for disaster preparedness.

How to use it?

Using the system is simple:

  • Navigate to the Predict Floods page.
  • Enter the required weather parameters (Temperature, Humidity, Cloud Visibility, Annual Rainfall, and Seasonal Rainfall).
  • Submit the form to receive an instant prediction detailing whether there is a High Flood Risk or a Low Flood Risk.

What methods do we use?

The core of our system is powered by robust Machine Learning techniques:

  • Dataset: We utilize a real-world meteorological dataset containing decades of weather records.
  • Preprocessing: The data is rigorously cleaned. Missing values are imputed using the median strategy, and outliers are neutralized using Interquartile Range (IQR) capping. The features are then normalized using a `StandardScaler`.
  • Model Training: Multiple algorithms (Decision Trees, Random Forests, KNN) were evaluated. Our current production model uses an XGBoost Classifier, which yielded the highest accuracy (~87%) for this specific dataset.
